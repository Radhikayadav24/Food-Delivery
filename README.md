# Food-Delivery# 🍔 Food Delivery Web Application

A full-stack **Food Delivery Web Application** built with the **MERN Stack**. Users can browse food items, add them to the cart, place orders, and make secure online payments. The project also includes an **Admin Panel** to manage food items and customer orders.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 🍕 Browse Food Categories
- 🛒 Add & Remove Items from Cart
- 💳 Secure Stripe Payment Integration
- 📦 Order Tracking
- 👨‍💼 Admin Dashboard
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Context API
- Axios
- CSS3

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)

### Payment
- Stripe

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Radhikayadav24/Food-Delivery.git
```

Move into the project directory:

```bash
cd Food-Delivery
```

---

## 2️⃣ Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd ../frontend
npm install
```

### Admin

```bash
cd ../admin
npm install
```

---

## 3️⃣ Configure Environment Variables

Create a **.env** file inside the **backend** folder.

```env
PORT=4000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

STRIPE_SECRET_KEY=your_stripe_secret_key
```

> **Note:** Never commit your `.env` file to GitHub.

---

## 4️⃣ Run the Project

### Start Backend

```bash
cd backend
npm run server
```

---

### Start Frontend

```bash
cd frontend
npm run dev
```

---

### Start Admin Panel

```bash
cd admin
npm run dev
```
